# KTALLER

Solución integral diseñada para la gestión eficiente de talleres de reparación. La aplicación permite digitalizar el flujo de trabajo, desde la recepción del equipo hasta la entrega final, optimizando el tiempo y la organización del negocio.

# Características

* **Gestión de Órdenes:** Creación, edición y seguimiento de órdenes de servicio en tiempo real.
* **Control de Estados:** Visualización clara del ciclo de vida de cada reparación (pendiente, en proceso, listo, entregado).
* **Tickets Digitales:** Generación automática de comprobantes de servicio.
* **Impresión Térmica:** Soporte integrado para impresión de tickets.

# Tecnologías Utilizadas

* **Lenguaje:** Kotlin
* **UI:** Jetpack Compose (Material 3)
* **Arquitectura:** MVVM con Clean Architecture
* **Persistencia:** Room Database
